Signature email Evan Orofino

Voici 3 versions de signature email à coller dans Gmail / Outlook. Cliquez-glissez pour sélectionner la signature, puis copiez (Ctrl+C).

📧 Pour Gmail : Paramètres ⚙️ → "Voir tous les paramètres" → onglet "Général" → section "Signature" → Créer → coller la version 1 ou 2 dans l'éditeur en mode formaté (pas en mode HTML brut).

📨 Pour Outlook : Fichier → Options → Courrier → Signatures → "Nouvelle" → coller dans l'éditeur.

📝 Bonus : Réglages bonnes pratiques

  • Mets une photo professionnelle dans ton profil Gmail / Outlook — elle apparaît à côté de tes mails et augmente le taux d'ouverture de 20%.
  • Configure une réponse d'absence en cas de RDV : "Je reviens vers vous sous 24h ouvrées. Pour les urgences, joignable au 06 XX XX XX XX."
  • Évite les emojis dans la signature si tu cibles des grosses boîtes (DSI, banques…). Garde uniquement le 📸 pour les photographes.
  • Mets ton numéro dès que tu seras à l'aise — c'est ce qui débloque les RDV chauds.
